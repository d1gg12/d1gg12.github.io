要写个wiresharks插件

发现用lua写挺不错的，而且lua的性能听说也很nb，可以写游戏

环境

windows下环境挺简单的，一个二进制包，然后path里填个路径就搞定了，然后就能直接vscode启动了。
由于这个是动态的脚本语言，所以vscode很舒服。

Linux下稍微有点烦，网上普遍的是下载源码自己编译，但是反正我ubuntu和kali都编译不通过，貌似还缺少什么依赖，不过apt里直接有
sudo apt-get install lua53也行，lua貌似没用通行性的版本（例如python2.7），所以基本上都带着版本号。

神必语法

  • # 这家伙是个一元运算符，返回字符串或表的长度，不太觉得这是个特别常用的功能hhh，做成一元运算符有点夸张。
  • ..用来连接字符串 ^用来乘方 只有这两个都是右连接的，从右到左运算
  • 除了''和""，[[aaa]]可以来写字符串，相当于python里的'''
